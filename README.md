# 🎯 Guess My Number Project

A simple “Guess My Number” game where the player tries to guess a randomly generated number. The application provides hints (too high / too low) to help the user reach the correct answer in as few attempts as possible.

This project was created to practice basic programming concepts such as conditional logic and user interaction.

# 🚀 Features

- Random number generation
- User input handling
- Instant feedback (too high / too low)
- Score tracking
- Highscore system
- Game reset functionality

# 🛠️ Technologies

- HTML
- CSS
- JavaScript

# 🎮 How to Play

1. Enter a number in the input field
2. Click the "Check" button
3. Read the hint and adjust your guess
4. Try to guess the correct number with the highest score possible
